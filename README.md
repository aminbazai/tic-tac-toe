> A modern, interactive Tic Tac Toe game with stunning animations and two exciting game modes!

## ✨ Features

- **Two Game Modes**
  - 👥 Player vs Friend (PvP)
  - 🤖 Player vs AI (Challenge the computer)

- **Smart AI Opponent**
  - Unbeatable AI using minimax-like logic
  - Strategic gameplay with blocking and winning moves

- **Beautiful UI/UX**
  - Smooth hover animations on cells
  - Glowing effects for X and O marks
  - Animated background orbs
  - Glassmorphism card design
  - Responsive layout for all devices

- **Score Tracking**
  - Live score display
  - New Round & Reset options

## 🚀 Getting Started

### 1. Clone the Repository
```bash
git clone https://github.com/YOUR_USERNAME/tic-tac-toe.git
```

### 2. Open in Browser
Simply open `index.html` in your favorite browser!

```
index.html
```

## 🎯 How to Play

1. **Select Game Mode**
   - Click "Me vs Friend" for 2-player mode
   - Click "Me vs AI" to play against the computer

2. **Make Your Move**
   - Click on any empty cell to place your mark (X or O)

3. **Win the Game**
   - Get 3 of your marks in a row (horizontal, vertical, or diagonal)

4. **Start New Round**
   - Click "New Round" to play again without resetting scores
